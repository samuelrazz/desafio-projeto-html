O que são Clients?

Clients

São dispositivos que consome as informações através da conexão HTTP com os servidores.

Esses dispositivos enviam solicitações aos servidores e estes retornanm com respostas arquivos tipo HTML, CSS, Javascript e imagens.

O navegador armazena dados de conexão que chamamos de cache para quando aquela pagina for acessada novamente, o navegador a carrega mais rápido, pois não necessita puxar toda a pagina do servidor, apenas as atualizações.

Para trabalhar com os Clients, é necessário considerar diferentes resoluções durante a criação de websites, chamado de Design Responsivo ou Responsividade, e também a compatibilidade entre navegadores.

O que são Servers?

Servidores

São máquinas que servem de apoio para outras máquinas da rede. Podem ser utilizadas tanto para internet quanto para rede local.

Outros tipos de Servidores

    Servidor Proxy: Serve internet para uma rede dentro de algumas regras
    Firewall: Camada de segurança externa
    E-mail Server: Servidor de E-mails
    Database Server: Servidores de Banco de Dados
    Servidor Web s/ internet: É um site que opera somente na rede privada de um sistema chamado intranet
    DNS (Domain Name Service): É um servidor que contém lista de nomes e IP's. Cada máquina tem um IP, que é como um RG da máquina na internet
    Hospedagem de sites: Significa colocar seus arquivos em um servidor na internet para os clientes acessarem. Pode ser contratado ou próprio
    FTP: É um tipo de conexão direta com o servidor para envio de arquivos
